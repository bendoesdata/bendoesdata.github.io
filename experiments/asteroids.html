This page streams live asteroid data every day from the NASA NeoWs (Near Earth Object Web Service). Each circle represents an asteroid in Earth's orbit. It's position in relation to blue sphere on the left is it's distance from Earth.

Distance is measured in lunar distance, a NASA measurement used to quantify space objects. One lunar distance is the distance from Earth to the Moon.

Hover over a circle to learn more about each asteroid.
